# GodotVirtualPC（in development）
Import a virtual pc into your game and create software for it.
##Built-in software that can be added or removed:
+ Email
  Can install，delete，receive or send emails.
+ Task Manager
  Can view computer processes.
+ Settings
  Can modify the computer's brightness and wallpaper.
+ Browser
  Allow developments to create web pages ,and players can enter by inputting the url.
